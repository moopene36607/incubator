## Customer Personas (k=3 clusters)

### 🧍 早班通勤族 — Cluster 0
- **時段中心**: 07:33(07:06-08:00)
- **客單價**: NT$ 79(NT$ 65-93)
- **n**: 60(30.0% of total)
- **營收貢獻**: NT$ 4,764

**客群描述**: 上班 / 上學前的快速早餐客;集中在 7-8 點之間 30 分鐘內,客單低且穩定(σ NT$13.8)。動機:**速度 + 經濟**,通勤路上順手買,沒時間挑。

**可能購買**:
- 三明治 / 飯糰 + 飲料(中冰 / 微糖)
- 蛋餅 / 鐵板麵套餐(NT$60-80)
- 美式咖啡 / 紅茶(NT$25-35)
- 點完就走,**外帶比例極高**

**促銷策略:早鳥 7:00-8:30「30 秒出餐套餐」**
- **細節**: 推 3 個固定品「早班快車套餐」(NT$80 / NT$95 / NT$110),提前在 6:30 預製,顧客 LINE 預訂可 8:00 前準時取餐;搭 LINE Pay / icash 一秒結帳
- **預期效果**: 提升早班 cluster 滿意度 + 客單向 NT$95 套餐升級 → 月營收 +NT$8K-12K

### 🧍 主婦 / 學生中餐 — Cluster 1
- **時段中心**: 11:32(10:54-12:10)
- **客單價**: NT$ 144(NT$ 119-168)
- **n**: 80(40.0% of total)
- **營收貢獻**: NT$ 11,522(**營收主力來源**)

**客群描述**: 中午 1 小時尖峰,客單高出其他 cluster **80%+**,是您店裡**營收主力 (58%)**。動機:**主餐 + 飲料 + 配菜 完整一餐**,內用比例高,願意花錢。可能含:① 附近主婦帶小孩外食 ② 大學生 / 上班族 ③ 老闆 / 經理請屬下吃飯。

**可能購買**:
- 套餐(主餐 + 飲料 + 小菜 / 湯),NT$120-160
- 義大利麵 / 燉飯 / 烤蔬菜
- 升級飲料 (鮮榨果汁 NT$80) 或加蛋 / 加肉
- 飯後加點 dessert(布丁 / 鬆餅)

**促銷策略:「11:00-13:30 升級套餐 +NT$30 = 飯後甜點 + 飲料 size 升級」**
- **細節**: 主餐固定價,套餐加價的升級包(飲料 size up + 1 個 dessert)成本約 NT$15 但定價 NT$30 = 100% 毛利;LINE 發午餐前推播「今日特餐 + 升級包」
- **預期效果**: 30% 中餐客戶選升級 = 80 客 × 30% × NT$30 = 額外 NT$720/週 × 4 週 = **月 +NT$ 2,900**

### 🧍 下午茶上班族 — Cluster 2
- **時段中心**: 15:35(14:58-16:12)
- **客單價**: NT$ 63(NT$ 50-76)
- **n**: 60(30.0% of total)
- **營收貢獻**: NT$ 3,807

**客群描述**: 下午 2:30-4:30 的「下班前小確幸」客,客單最低(NT$63),通常單品(飲料 + 點心 二選一)。動機:**情緒紓壓 + 小休息**,可能是:① 附近辦公室上班族下午茶 ② 全職媽媽接小孩前的喘息 ③ Freelancer 換個環境工作。

**可能購買**:
- 鮮奶茶 / 冰拿鐵(NT$45-65)
- 鬆餅 / 蛋糕單片(NT$60-90)
- 偶爾搭餐(NT$120 加價套餐)
- 內用比例中等(可能會待 1-2 小時)

**促銷策略:「14:00-16:30 第 2 杯飲料半價 + 集點 5 點換鬆餅」**
- **細節**: 第 2 杯半價鼓勵雙人 / 朋友來;集點貼紙 5 點(即 5 次下午茶)換 1 個鬆餅 → 用「下個月再來」的承諾留客
- **預期效果**: 第 2 杯半價拉雙客率從 20% → 40% = 客單從 NT$63 → 約 NT$90 = 月 +NT$ 6K-8K;集點促回訪率上升

## 整體經營洞察

- **Cluster 1(主婦學生中餐)是您的營收核心(58%)— 必須優先穩固**:中餐 80 筆只佔筆數 40% 但貢獻 58% 營收;若中餐單流失 10%,等同您**整月營收掉 5.8%**。建議:① 中餐尖峰增加 1 個 part-time 加快出餐 ② 推「中餐 VIP 集點卡」鼓勵每週 ≥ 2 次回訪 ③ 中餐主餐項目擴張(目前可能太集中在 2-3 道)
- **Cluster 2(下午茶)客單最低但量大,是「升級空間最大」的 segment**:NT$63 升到 NT$90 = +43%。每月 60 客 × NT$27 升級 = NT$1,620。**透過套餐 / 集點 / 第 2 杯半價**可以做到。下午茶若能升級,**整體營收提升 10%+**(從 NT$20K → NT$22K)
