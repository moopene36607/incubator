## ⚠️ Hierarchical Bayes 模型假設與限制

- **常態假設**: rating ~ Normal, 真實是 1-5 ordinal -- Pro 版用 ordinal probit 較精確
- **僅 2-level shrinkage**: (cell ← global), 實務上 (品種 ← 屬科 ← 全國) 三層更穩;Pro 版加品種分群
- **σ_within / τ_between 用 method-of-moments**: 不一定 MLE 最優;大樣本 OK, n_cells < 20 時改用 REML
- **未考慮飼主主觀偏差**: 同一隻狗不同主人打分差 0.5-1.0 顆星, Pro 版加 reviewer-effect random intercept
- **冷啟動弱**: 新品種 / 新品牌 0 review 時退回 μ_global, 仍鼓勵實地試吃 + 寵物醫師意見
- **不取代獸醫專業**: 嚴重腸胃 / 過敏 / 慢性病請先諮詢獸醫, 飼料只是輔助

---
*petfeed = Empirical-Bayes Hierarchical Linear Model (James 1961 / Efron-Morris 1973) × 台灣寵物飼料 niche = 把零碎 (品種, 品牌) 評分 shrink 成穩健後驗推薦, n=2 的 5.0 不會打敗 n=30 的 4.6, 飼主一次選到 sample-bias 小的飼料.*

## 🤖 AI 寵物營養顧問建議

陳太太家貴賓 6 歲 4.5 kg 腸胃敏感, 直接推 Royal Canin 皇家。理由是它在貴賓品種有 8 件 review 佐證, 後驗 4.66 幾乎沒被模型修正, 是「真的好」而不是「剛好評到 5 分」; 反觀 ZiwiPeak 雖然數字最高 4.92, 但只有 2 個飼主的數據 -- 在敏感腸胃情境風險偏高, 第一次不建議冒。Orijen 是無穀備胎 (4.59), 萬一皇家不適合可換它。

換糧 SOP 給陳太太說清楚:第一週舊飼料 75% + 皇家 25%, 第二週 50/50, 第三週 25/75, 第四週全換。每天觀察便便 (軟便 / 黏液 / 血絲) 與食慾, 出現持續軟便 3 天以上立刻停換並回診。皮膚搔癢 / 耳朵抓 / 眼周流淚 = 過敏跡象, 退回原飼料並安排獸醫過敏原檢測。

寵物店 SKU 採購邏輯: 皇家 / 希爾思 / Orijen 三家在貴賓品種 n≥4 後驗穩定, 主推位置擺出來; ZiwiPeak 雖然後驗高但 n=2 屬於試點 -- 進小包試吃裝、不囤大袋; 寶路在貴賓 / 柴犬 / 米克斯三品種都被打到 3.0-3.4, 即使利潤好也應該退出主推, 改放促銷區處理庫存。冠能 1 件 n=1 不夠信號, 觀察期。

風險提醒: 陳太太敏感腸胃描述若併發嘔吐、體重下降, 屬於 IBD / 胰臟炎可能, 不是換飼料能解決, 請先帶去做血液 + 糞便檢查再決定要不要走處方飼料 (i/d 或 GI Low Fat) 路線。
