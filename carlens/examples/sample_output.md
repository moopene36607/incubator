## 故事化解釋(AI 顧問)

老闆,這台車**表面看是「2018 年 4 萬公里一手車」的完美物件**,但 4 個獨立來源的證據組合起來,**強烈暗示這是一台「事故修復 + 調表 + 可能泡水」的拼裝中古車**:

1. **調表嫌疑**:行照寫 3.8 萬 km,但內裝磨耗(方向盤露白、椅墊塌陷、踏板見金屬底、檔位字樣模糊)= 典型 8-10 萬 km 的車況。賣方可能將真實里程約 **8-10 萬 km 調回 3.8 萬 km**。
2. **前方撞擊修復**:引擎室「局部重噴」+ 右前葉子板螺絲鎖過痕跡 + 水箱罩支架疑似換新 = 前方碰撞修復典型徵兆,**與『無事故』宣稱直接矛盾**。
3. **疑似泡水**:底盤整體新塗裝(看不到原廠焊接點)+ 邊緣滴流 = 用塗裝**掩蓋底盤生鏽或泡水痕跡**,**與『無泡水』宣稱矛盾**。
4. **板金歷史**:右後車門 / 葉子板色差 + 後保桿 1mm 高低差 = 後方板金修補。

組合起來,這台車最可能的真實歷史是:「8-10 萬公里高里程運將 / 業務車 → 經歷前後雙向事故 → 可能泡水或長期淋雨 → 整新後重新進入中古車市」。**車身號碼牌雖然原廠真實,但車內外多處不一致**。

## 風險訊號 (純函式計算)

### 🔶 [HIGH] `MILEAGE_TOO_LOW_FOR_WEAR` (+25)
- 行照里程 38,000 km 對應的合理內裝磨耗區間為 2-4/10,但實際觀察內裝磨耗 6/10,**顯著高於預期**。這是**調表車最典型的徵兆** — 賣方可能將實際里程從 ~80,000+ km 調回 38,000 km。

### 🔶 [HIGH] `ENGINE_BAY_CONTRADICTS_NO_ACCIDENT` (+15)
- 賣方宣稱『無事故』,但引擎室狀態為「局部重噴」。引擎室重噴通常代表前方碰撞修復或結構件矯正。**這與『無事故』直接矛盾**。

### 🔶 [HIGH] `UNDERCARRIAGE_FRESH_PAINT_SUSPICIOUS` (+20)
- 底盤『新塗裝可疑』 — 中古車底盤重新塗裝通常是為了掩蓋生鏽 / 泡水痕跡。與『無泡水』宣稱矛盾。

### ⚠️ [MEDIUM] `BODY_PAINT_PARTIAL_MISMATCH` (+10)
- 車身漆面『局部色差』 — 常見於板金修補。輕度修補不一定是事故,但需要賣方說明清楚。

## 最終建議

- **決策**: **不建議購買**
- **理由**: 4 個獨立來源的證據組合起來證據力極強(調表 + 事故 + 疑似泡水 + 板金),即使議價也無法消除結構性風險。Camry 是熱門車款,**8891 上同款乾淨車選擇很多**,不必為這台冒險。
- **具體條件**(若仍堅持購買):
  - 議價底線:**NT$ 380,000**(以實際車況推估)
  - 簽約必加條款:**「賣方擔保無事故 / 無泡水 / 真實里程,如不實買方有權無條件退款 + 賠償 NT$ 100,000」**(沒有這條的賣方,根本不敢簽)
  - **必先帶到 Toyota 原廠做完整檢查 + 出具書面報告**(NT$ 3,000-5,000)
  - 7 天無條件退款條款

## 進一步盡職調查清單

- **Toyota 原廠完整檢查**:讀取車輛 ECU 內**最大記錄里程**(調表後 ECU 仍會記錄,賣方無法竄改)+ 排氣管 / 引擎室 / 底盤完整檢查 NT$ 3,000-5,000
- **VIN 車身碼鑑識**:到監理站查行照、車身碼、引擎號碼三者一致性
- **司法院車籍查詢**:https://www.thb.gov.tw → 是否有事故車登記
- **強制險紀錄查詢**:財團法人保險事業發展中心 → 過去是否有理賠紀錄
- **同款車市價比對**:8891 / 比比看 / Yahoo 拍賣 → 同年 / 同公里 / 同色比較(若超過市場 +10% 直接放棄)
- **車主住址歷史**:若有可能,問賣方為何「出國移民急售」 — 真實狀況的車主多半願意提供護照 / 居留證影本佐證
- **詢問前車主**:行照通常會列前車主資訊,可嘗試聯絡確認車況歷史 / 是否有事故 / 真實里程
- **車輛照片倒查**:用車牌 / VIN 在 Google / 8891 過去 listing 倒查,確認是否曾以不同說法上架(常見「事故重組車多次掛賣」)
