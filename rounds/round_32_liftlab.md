### Round 32 — `liftlab/` (Taiwan 🇹🇼 — SME marketing causal ROI via Pearl backdoor adjustment, **twenty-second AI pattern — Causal Inference / Pearl do-calculus**)

- **題目**: 台灣 SME 行銷投放真實因果 ROI 評估(24+ 月廣告 + 營收資料 → 純函式 Pearl backdoor adjustment 控制 confounders[旺季 / 節慶 / 新品]→ 切出純廣告因果效應 → LLM 故事化解釋 + 真實 ROI 計算 + action items + caveats)
- **解決的問題**: 5-30 人 SME 每月投 NT$30K-200K 廣告(IG / FB / Google),老闆**用 naive ROI = (高 ads 月 - 低 ads 月) / 廣告費** 算完全忽略 confounders。Dcard 創業板「6 個月每月投 5 萬廣告營收成長 30%,但停廣告 1 個月只跌 10% 誰才對?」;FB 老闆「代理商給我 ROAS 7x 報告,自己算月底剩多少現金根本沒多賺」;PTT marketing「Google Analytics 不會跟我說『多少是季節』」。老闆通常把廣告**剛好投在旺季 / 大檔 / 新品月**(selection bias)→ 那些月份本來就會多賺 → **naive ROI 高估 2-3 倍**。Google Analytics / Mixpanel 提供 last-click attribution **不做因果調整**;HubSpot lead-based 不控制 confounders;代理商月度報告**有利益衝突**(要你續約)用 naive ROAS;Excel 算 backdoor adjustment 公式繁雜老闆沒統計背景;R / Python causal-inference 套件(DoWhy / CausalNex)完全給研究員;學術論文 / 顧問報告一次 NT$30-100K;ChatGPT 不能跑 stratified ATE 每次重貼資料。**Gap 結構性**:Pearl 因果推斷學術界成熟 30 年,但**沒有人做成 SME 老闆可用的 SaaS**。Google「台灣 行銷 因果 ROI」零中文 SaaS。架構:**Causal Inference / Pearl do-calculus** — `causal.py` 100% 純函式 stdlib(只用 statistics + dataclass + itertools);① compute_naive_ate = E[Y|T=1] - E[Y|T=0](老闆通常算法)② compute_backdoor_adjusted_ate(Pearl 1995):按 (peak, holiday, new) 分 8 個 strata,在每個 stratum 內計算 ATE,加權平均 ATE = Σ_z [E[Y|T=1,Z=z] - E[Y|T=0,Z=z]] × P(Z=z)③ compute_confounder_bias 拆解 naive vs adjusted 由哪個 confounder 貢獻多少;④ inflation_factor + verdict 純函式產出。LLM 寫 200 字故事化 exec summary + confounder 翻譯(把「is_peak_season bias +NT$109K」翻成「旺季本身會自帶 NT$120K 營收」)+ 真實 ROI 計算(adjusted ATE / avg ad spend × 毛利率)+ 4-5 條 action items + caveats。**LLM 永不算 NT$ / ATE**、**誠實 caveats**(只控 3 個 confounders;真實還有對手 / 經濟 / 天氣 / 媒體 / 口碑)、**小樣本警示**(< 24 月)。Sample: 24 月合成資料(true ATE = NT$80K),naive 估出 NT$190K (2.3x inflated),adjusted 估出 NT$83K(完美還原真值)。
- **目標市場**: 5 萬投廣告核心 SME。Free 1 次/月(≤14 月)、Solo NT$499/月(1-5 人,3 channel + 24-48 月)、Pro NT$1,499/月(5-30 人 + 多 channel + sensitivity + LINE)、Enterprise NT$4,999/月(30+ + Google Ads / Meta Ads API)、B2B 代理商 white-label NT$8,000/月、B2B 顧問 NT$30K+/月。WTP 錨點:行銷顧問做一次分析 NT$30-100K vs Pro NT$1,499 = **20-66x 便宜**;老闆過去以為加碼 NT$120K/月廣告,liftlab 揭示真實 ROI 1.05x → 省 NT$40K/月浪費 budget = **單月 27x 回本**;代理商 NT$8K/月 給 30 客戶 = 每客戶 NT$267 收 NT$2-5K 很划算。TAM 1% 滲透 = 500 家 × NT$1,499 = **月 NT$75 萬 MRR / 年 NT$900 萬 ARR**;加 Enterprise + B2B 代理商 + 顧問 → 年 ARR NT$2,000-4,000 萬;橫移日韓港新 → 翻倍。Distribution: FB 中小企業老闆社群、PTT marketing / Soft_Job / SOHO、Threads / IG SME KOL、行銷代理商 partner(凱絡 / Foundry)、數位時代 / Inside / 經理人合作、Google Ads / Meta 訓練課程 partner、創業育成中心 / 商總。
