### Round 31 — `teachsay/` (Taiwan 🇹🇼 — K-12 teacher–parent LINE communication AI assistant with style learning, **twenty-first AI pattern — Active Learning / Human-in-the-loop**)

- **題目**: 台灣國中小老師家長 LINE 溝通 AI 助手 + 個人風格 active learning(過去 5-10 回覆 corpus → 純函式 12 特徵 style profile → 新訊息 intent 分類 → LLM 生成 3 個 tone 草稿[formal / warm / brief]+ style match score → 老師選 + 微調 → 純函式 weighted-average update profile → 下次更準)
- **解決的問題**: 台灣國中小老師約 11 萬 + 補習班 / 才藝班 5 萬 = 16 萬,**每位每月 200-500 則 LINE 訊息**,每則 2-5 分鐘 = **每月 10-40 小時純文字工(隱形加班)**。Dcard 教師版「下班 9 點還在回家長 LINE 週末更慘」;FB 教師交流「ChatGPT 寫的太罐頭家長一看就知道不是我寫的」;PTT C_Education「同樣請假訊息每天 5 次但每次重打因為要個人化」。多數訊息重複 patterns(請假 / 作業 / 行為 / 行程 / 抱怨)但需個人化;ChatGPT 直接生成**太罐頭**(「您好已收到會處理」家長反感);通用 AI 不認識老師個人風格(有的愛驚嘆號 / 有的愛敬語 / 有的短促);LINE 罐頭回覆完全沒個人化;Cogniti / Quillbot 海外英文不認識台灣家長文化(輩分 / 敬語 / 親近度)。**Gap 結構性**:中文教師 + LINE 溝通 + 個人風格學習零 SaaS。Google「台灣 老師 LINE 家長 AI 助手」零結果。架構:**Active Learning / Human-in-the-loop** — `style.py` 100% 純函式抽 12 個風格特徵(平均字數 / 開頭暖度 / 結尾正式度 / emoji 密度 / 敬語密度 / 語氣詞 / 引用具體事件 / 具體 action 次數 / 段落 / 問句 / 驚嘆密度);`intent.py` 純函式 keyword-based 6 類分類(leave_request / homework / behavior / schedule / complaint / general)+ urgency(行為 / 抱怨 → high);Claude 生成 3 個 tone alternatives(formal / warm / brief)每個帶 style_match_score 0-100;老師選 + 微調 → 純函式 weighted-average update_profile_with_new_sample(weight = 1/(n+1) 讓早期樣本影響大快速適應後期穩定收斂);純函式 compute_diff 找 top 5 風格變化;LLM 用 50-100 字解釋「學到什麼」。**長期 60+ 樣本後 match 達 95+**。**LLM 永不算 style_match_score**、**behavior / complaint 自動拒生草稿強制老師手寫**、**最終訊息送出前老師審閱**。
