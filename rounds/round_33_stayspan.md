### Round 33 — `stayspan/` (Taiwan 🇹🇼 — SME employee retention survival analysis with Kaplan-Meier, **twenty-third AI pattern — Survival Analysis / Time-to-Event**)

- **題目**: 台灣 SME 員工 retention 時間分析(employee history CSV → 純函式 Kaplan-Meier 留任曲線 + median tenure + 12/24/36 月留任率 + 各部門 / 職級 / 績效 cohort 比較 + hazard ratio → LLM HR 顧問寫 5 條留任行動 + 早期警示)
- **解決的問題**: 5-30 人 SME 每年招 3-10 人,每位離職成本 NT$50-100K(招聘 + 培訓 + 上手期失能)。Dcard 創業板「招了工程師月薪 6 萬做 3 月跳槽,18 萬白花」;PTT Soft_Job「想知道我這行業平均員工做多久」;FB 老闆「104 / 1111 給薪資但不告訴我員工平均做多久」。**104 / 1111 給薪資水準不做 retention 預測**;BambooHR / Workday 偏大企業英文 UI 複雜;EZHR / 行動人事 是排班 / 打卡不做 retention;HR 顧問報告一次 NT$30-50K;R / Python survival 套件(lifelines / scikit-survival)完全給研究員;Excel 算 KM 公式繁雜;ChatGPT 不能處理 censored data。**Gap 結構性**:Kaplan-Meier 1958 年就有但 60+ 年來沒人做成 SME HR 可用 SaaS。Google「台灣 中小企業 員工 留任 預測」零中文 SaaS。架構:**Survival Analysis / Time-to-Event** — `survival.py` 100% 純函式 stdlib(只 statistics + dataclass);① compute_kaplan_meier 處理 right-censored data S(t) = ∏(1 - d_i / n_i) for event times t_i ≤ t ② median_tenure 首個 S(t) ≤ 0.5 的時間 ③ survival_at 查詢 S(12), S(24), S(36) ④ cohort_survival 按部門 / 職級 / 績效跑 KM 比較 ⑤ compare_cohorts 找最危險 vs 最穩定 cohort + simplified hazard ratio。LLM 寫 150-200 字 executive summary + 最危險 cohort + 最穩定 cohort + 4-5 條留任行動(優先順序 + 預估省下的離職成本)+ 早期警示信號 + caveats。**LLM 永不算 NT$ / 留任率**、**Cohort 級別而非個體**(報告強調避免「用 hazard 開除員工」)、**績效 tier 是預測力最強但可能有 reverse causality**(需 causal analysis 確認)、**小樣本警示 < 30 員工結論信賴度有限**。Sample: 30 員工(17 已離 + 13 在職)→ median 22 月、S(12)=73.3%、Engineering 最危險 S(12)=60% median 15 月、Ops 最穩定 S(12)=100%、Low 績效 8 人全離職 median 8 月 vs High 績效 11 人只走 1 人。
- **目標市場**: 5 萬 5-30 人有 HR 概念的核心 SME。Free 1 次/月(≤20 員工)、Solo NT$799/月、Pro NT$2,499/月(30-100 人 + Cox PH + 介入追蹤)、Enterprise NT$8,999/月(100+ + HRMS API + 個人化 hazard 僅 HR 看)、B2B 人力顧問 NT$30K+/月、B2B 獵頭 NT$50-200K/月。WTP 錨點:HR 顧問 NT$30-50K vs Pro NT$2,499 = **12-20x 便宜**;一次擋下「招到 6 月就走的人」NT$60-80K vs Solo NT$799 = **單次擋下回本 6 年**。TAM 1% 滲透 = 500 家 × NT$799 = **月 NT$40 萬 MRR / 年 NT$480 萬 ARR**;加 Pro + Enterprise + B2B 顧問 / 獵頭 → 年 ARR NT$2,000-4,000 萬;橫移日韓港新 → 翻倍。Distribution: PTT Salary / 中小企業 / Soft_Job、FB 中小企業老闆社群、HR 社群 / SHRM 台灣、HR 顧問 partner、創業育成中心 / 中小企業協會、YouTube / LinkedIn HR KOL、104 / 1111 整合、Cake / Yourator 招募平台 integration。
