### Round 27 — `bizradar/` (Taiwan 🇹🇼 — SME client / supplier risk assessment via public-data graph analysis, **seventeenth AI pattern — Graph / Network Analysis + Entity Resolution**)

- **題目**: 台灣中小企業客戶 / 廠商風險評估(SME 版 D&B)— 統編 / 公司名 → 純函式 graph 分析公開資料(商業司 + 司法院 + 新聞)→ 算 risk_score → LLM 寫故事化解釋 + 合作條件建議(預收 % / 票期 / 違約金 / 個人連帶保證 / 信用保險)
- **解決的問題**: 5-30 人 SME 簽 NT$50-500 萬合約前**只能憑感覺或鄉里口碑判斷對方會不會倒帳**。Dcard 創業板「跟新客戶簽 200 萬大單發現他董事 1 年前剛把另一家公司倒掉」;PTT Salary「D&B 一查 NT$30,000 起跳,我們連 200 萬合約毛利都才這個數」;FB 中小企業老闆社群「自己查商業司、司法院、Google 新聞,1 個客戶 1-2 小時」。**D&B NT$30K+ 起跳**只服務大企業評大客戶;TEJ 偏上市櫃對 SME 涵蓋低;聯徵中心只能查自己不能查客戶;商業司公開資料 + 司法院裁判書 + 新聞**資料公開但散在 3-5 個網站無 graph 連結**;ChatGPT 容易幻覺;徵信社一案 NT$5-15K 耗時 3-7 天且不擅長「金蟬脫殼」型風險(舊公司倒掉甩債馬上開新公司用相似名字 — 沒有 graph 分析根本看不出來)。**Gap 結構性**:公開資料完全足夠,但**整合 + entity resolution + graph 分析**沒人做。Google 搜「台灣 中小企業 客戶 風險 評估」零 SaaS 結果(D&B 廣告除外)。架構:**Graph / Network Analysis + Entity Resolution** — `graph_analyzer.py` 100% 純函式:① find_director_connections 找目標公司董事的所有任職紀錄(reverse-index director→companies)② find_address_collisions 查共用地址 ③ check_director_problem_history 判斷董事是否任職過已解散 / 多訴訟 / 負評公司;risk signals 8 類(NEW_COMPANY +25, LOW_CAPITAL +20, SHARED_ADDRESS +15, DIRECTOR_LINKED_TO_INACTIVE +25 per 公司, DIRECTOR_LINKED_TO_LAWSUIT_HEAVY +15 per 公司, RECENT_DEFENDANT +12 per 件, NEGATIVE_NEWS +5-10 per 篇, BUSINESS_INACTIVE +80 強制 CRITICAL);分檔 LOW (≤25) / MEDIUM (≤50) / HIGH (≤75) / CRITICAL。LLM 寫 100-200 字故事化解釋(關鍵:把 graph 上看到的「鴻禾 ↔ 禾鴻」金蟬脫殼模式串成故事)+ 5-7 條合作條件建議(預收 % / 票期 / 上限 / 違約金 / 個人連帶保證 / 信用保險)+ 進一步盡職調查清單。**LLM 永不算 risk_score**、**嚴禁完全拒絕往來建議**(留給老闆判斷)、**只用公開資料避免個資爭議**。
- **目標市場**: 150 萬 SME + 50 萬有對外合作需求。Free 3 次/月(無 AI 解釋)、Solo NT$299/月 + 30 次 AI 解釋、Pro NT$999/月 + 200 次 + LINE 推播、Pay-per-use NT$199/次、B2B 會計師 / 律所 NT$3,999/月(批量上傳 1,000 客戶)、B2B 銀行 / 徵信 white-label NT$50-200K/月、API NT$3/call。WTP 錨點:徵信社一案 NT$5-15K vs Pay-per-use NT$199 = **25-75x 便宜**;一次「沒查就簽合約」被倒帳 NT$50-500 萬 vs Pro NT$999/月 = **單次預防回本 5-500 年**。TAM 1% 滲透 = 5,000 家 × NT$999 = **月 NT$500 萬 MRR / 年 NT$6,000 萬 ARR**;加 B2B 會計師 / 律所 / 銀行 / 徵信 → 年 ARR NT$1-3 億;橫移日韓港新 → 翻倍。Distribution: PTT Salary / 中小企業板、FB 中小企業老闆社群、記帳士 / 會計師事務所 partner、創業育成中心 / 商總、YouTube 中小企業 KOL case study、LinkedIn B2B 開發、TEJ / 中華徵信所 partner、商業司 hackathon 露出。
