### Round 39 — `peakflow/` (Taiwan 🇹🇼 — restaurant peak-hour agent-based simulation, **twenty-ninth AI pattern — Agent-based Discrete-Event Simulation**)

- **題目**: 台灣中小餐廳尖峰時段營運模擬器(Customer / Server / Kitchen 三類 agent + event-driven discrete-event simulation + Monte-Carlo 多 scenario 比較 → AI 為已算好的 KPI/ROI 寫老闆讀得懂建議)
- **解決的問題**: 台灣餐飲業 17 萬家,5-30 人小餐廳佔 50% = 85,000 家。午餐 / 晚餐尖峰決定生死,但老闆做擴張決策**沒有工具**:「加 1 個服務員 NT$35K 真的有用嗎」、「擴廚 vs 推快餐套餐哪個賺」、「加桌位 vs 客人等到走」每週 PTT food / Dcard 餐飲業 / FB「餐飲老闆交流」5-10 萬人社群重複問。iCHEF / 沛點 POS 是歷史報表**不模擬**假設情境;餐飲顧問 1 次 NT$10-50K + 主觀經驗;Excel 預估表不會跑 Monte-Carlo;AnyLogic / Simul8 USD$1,500+ 月費學習陡;海外 Restaurant Sim 不認識台灣翻桌文化 / 客單價。Discrete-event simulation 1960 年代成熟,**沒人做成台灣餐廳老闆可用 SaaS**。架構:**Agent-based Discrete-Event Simulation** — `sim.py` 100% 純函式 stdlib(heapq + random + statistics + dataclass);Customer agent: Poisson arrival + party_size (1-4 加權 20/50/15/15) + Gaussian patience + 8 狀態 state machine;Server agent: N 個外場 each with `busy_until` 時間軸;Kitchen agent: capacity-bounded FIFO order queue;Event queue heapq 排序處理 arrival → seat → take_order → send_to_kitchen → food_ready → deliver → finish_meal → free_table;`_find_table()` smallest-fitting packing 避免 4 人坐 6 人桌;`run_scenarios()` Monte-Carlo 每 scenario 跑 N 次平均。LLM 只為 KPI/ROI 寫 5-7 行老闆建議(指出瓶頸 / 推薦方案 / 1-2 個風險)。**LLM 永不算營收 / 流失率 / ROI**(100% 純函式)、**永不改純函式結果**。Sample: 阿明小館午餐 48 組進店 → 現況流失 36.5% → **發現真瓶頸是廚房 84.7% 滿載** → 推快餐套餐 ROI +1125%(月淨 NT$+135K) vs 全升級 ROI +345%(月淨 NT$+199K 但砸 NT$58K 固定成本)→ 教老闆「先動菜單再砸人力」。12 edge tests passed(monotonic capacity、deterministic seed、kitchen capacity 不超過上限、revenue only from served)。
- **目標市場**: 85,000 家 5-30 人小餐廳。Solo NT$799/月(1 家店、3 情境/月、10 runs/sim)、Pro NT$1,999/月(5 店、無限情境、50 runs、POS 匯入)、Enterprise NT$8,000+/月(連鎖 + iCHEF/沛點 API)。WTP 錨點:1 次餐飲顧問訪談 NT$10-50K vs Solo NT$799 = **12-62x cost saver**;老闆做錯 1 次「擴廚 vs 加人」沉沒成本 NT$30-100K → 1 個月年費就回本。TAM 5% × 85,000 = 4,250 家 × Solo NT$799 = **月 NT$340 萬 MRR / 年 NT$4,080 萬 ARR**;加 Pro + Enterprise + 顧問白標 → 年 NT$8,000 萬 ARR 天花板。Distribution: PTT food / restaurant、Dcard 餐飲業、FB「餐飲老闆交流」5-10 萬人社群案例分享、iCHEF / 沛點 / Foodland POS partner(20% 拆帳)、餐飲顧問公司白標(月 NT$5K)、商業周刊 / 食力 foodNEXT 撰文、台北國際食品展(每年 6 月)。
