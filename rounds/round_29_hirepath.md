### Round 29 — `hirepath/` (Taiwan 🇹🇼 — SME "hire-vs-outsource" full-lifecycle ROI decision support, **nineteenth AI pattern — A/B Decision Modeling with Uncertainty Bands**)

- **題目**: 台灣 SME 老闆「招專員 vs 外包」12/24/36 個月全週期 ROI 對照(profile JSON → 純函式精算雇用 hidden cost[健保 5.17% / 勞保 10.5% / 勞退 6% / 三節 / 年終 / 招聘 / 培訓 / 設備 / 重招風險] + 外包 hidden cost[管理時間 / 切換風險] → LLM fractional COO 顧問評估 4 個 non-cost factors[品質 / 速度 / 機密性 / 規模化] + 給最終建議招 / 外包 / 混合)
- **解決的問題**: 5-30 人 SME 老闆**每年面臨 3-5 次「招專員 vs 外包」決策**(行銷 / 設計 / 內容 / 客服 / 業務 / IT 維運),憑直覺決定常常事後懊悔。Dcard 創業板「招了 Marketing 月薪 5 萬以為一年 60 萬,結果加上勞健保 + 三節 + 年終 + 設備 + 6 月離職重招,一年花了 100 萬什麼都沒留下」;PTT Soft_Job「外包代操月費 3.5 萬看似便宜,但廠商同時做 10 家同行,我們最大競爭對手就是他們客戶」。**雇主隱性成本**(健保 5.17% + 勞保 10.5% + 勞退 6% + 三節 + 年終 + 招聘 + 培訓 + 設備)= 月薪 × 1.6-1.8 倍 SME 老闆**常常只算月薪**;**外包隱性成本**(老闆對接時間 + 廠商切換 + 知識遺失)= 月費 × 1.2-1.5 倍常被忽略。會計師看「過去 P&L」**不做未來 12-36 月全週期 ROI**;HR 顧問偏服務大公司不接 SME;Mercer / KPMG 報告 NT$50K+ 一份;ChatGPT 算不出健保勞保提撥 / 三節 / 重招風險;Excel 自己算公式繁雜(NHI cap NT$182K / LI cap NT$45.8K / 勞退 cap NT$150K 各不同)。**Gap 結構性**:SME 每年 100 萬次招募決策但**沒有任何在地化、輕量、整合「成本 + non-cost factors」的決策工具**。Google「台灣 中小企業 招專員 外包 決策」零 SaaS 結果。架構:**A/B Decision Modeling with Uncertainty Bands** — `comparator.py` 100% 純函式(NHI 5.17% × cap / LI 10.5% × cap / 勞退 6% × cap / 三節 + 年終 = 月薪 × 2.5/年 / 招聘 / 培訓 / 設備 / 重招風險);**雇用 effective cost = 月薪 × 1.6-1.8 倍** 明確列出;`compare` 兩方案總成本對照 + 月化節省 + breakeven horizon + qualitative_signals(高薪 / 高 mgmt 時間 / 高離職率自動 trigger 警示);`horizon_sensitivity` 跑 1/2/3/5 年看趨勢。LLM 評估 4 個 non-cost factors 各 1-5 分(品質 / 速度 / 機密性 / 規模化)+ 最終建議**招專員 / 外包 / 混合**(很多 case 答案是「短期外包驗證 12 月 + 第 13-18 月內化」)+ 5 條決策觸發條件(業務量翻倍 / 廠商漲價 / 公司擴張等)。**LLM 永不算 NT$**、**不勸無條件選便宜的**、**3 個 ending 識別混合策略**。
- **目標市場**: 150 萬 SME + 30 萬 5-30 人核心。Free 1 次/月、Solo NT$299/月(3 次完整 AI + 比較記錄)、Pro NT$999/月(無限試算 + 多人團隊)、Pay-per-use NT$499/次、B2B 會計師 / 顧問 NT$3,999/月 white-label、B2B 創投 / FA NT$30K+/月。WTP 錨點:會計師客製試算 NT$5-30 萬 vs Pro NT$999 = **5-30x 便宜**;一次「錯招」損失 NT$50-100 萬,NT$299/月 預防;B2B 會計師事務所每年 50-100 次 SME 客戶該類討論,Pro NT$999/月 vs 自己手算 50-100 hr × NT$1,500 = **75-150x ROI**。TAM 1% 滲透 = 3,000 家 × NT$999 = **月 NT$300 萬 MRR / 年 NT$3,600 萬 ARR**;加 B2B 會計師 + 創投 → 年 ARR NT$1-2 億;橫移日韓港新 → 翻倍。Distribution: PTT Soft_Job / 中小企業 / Salary、FB 老闆社群、記帳士 / 會計師事務所 white-label partner、創業育成中心 / 商總、YouTube SME KOL、LinkedIn B2B、HR 顧問轉介、iCHEF / Cashier integration。
