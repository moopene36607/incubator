## ⚠️ Naive Bayes 模型假設與限制

- **獨立性假設**: P(features | class) = Π P(feature_i | class) — 假設各特徵獨立, 真實有相關 (e.g., 老人 + 嬰兒共存); Pro 版用 TAN (Tree-Augmented NB)
- **Laplace 平滑 α = 1.0**: 對 unseen feature value 給最小概率, 但仍可能 over-smooth
- **訓練樣本 100 件不足**: 真實 launch 需 ≥ 500 件多樣 pairing
- **類別不平衡**: 老人照顧 friendly 只 4 件 → 信心區間寬, 容易誤判;Pro 版加 class weight 補正
- **不取代實地評估**: 推薦 specialty 是初步分類, 具體阿姨還需老闆 / 平台二次媒合
- **隱私敏感**: 客戶家庭資料 (老人 / 嬰兒 / 寵物) 涉個資, 雲端版需加密 + 客戶同意 + 資料留存政策
