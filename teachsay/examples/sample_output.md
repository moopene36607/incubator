# teachsay 老師家長 LINE 溝通助手報告

**模式**: 純函式 style + intent + LLM 草稿 + active learning

## 老師資料

- **老師**: 陳老師
- **學校 / 班級**: 新北市某國小四年三班
- **歷史回覆學習樣本**: 6 則

## 新進家長訊息

> 老師您好,小明明天因為要去看牙齒醫生(早上 9:30 預約),早上會晚一點到學校,大概 10:30 進教室。麻煩您讓小明先進來,我會跟他說要先看一下今天上課進度。謝謝!

- **Intent**: `leave_request` (請假)
- **Urgency**: `high`

## AI 生成草稿(3 個 tone)

### Tone: **formal** (style match 78/100)
```
家長您好,謹此告知:小明明天 10:30 將進入教室,我會請他到座位坐定後即融入課程。上午第一節為國語,第二節為數學。為避免他遺漏教學內容,午休時間我會用 5 分鐘為他補上重點。請家長協助簽具請假單後讓小明帶來,如後續牙齒處理時間有調整請即時告知。
```
**為什麼這樣寫**: 結尾用「謹此告知」「請即時告知」較正式,適合不熟的家長 / 第一次互動。但保留陳老師標誌性的「家長您好」開頭 + 「具體上午課程」+ 「午休 5 分鐘」action item。

### Tone: **warm** (style match 89/100)  ⭐ 推薦
```
家長您好,沒問題的!那小明明天 10:30 進教室,我會請他直接到座位上課。上午第一節是國語、第二節是數學,我會在午休時找他 5 分鐘把上午的重點補給他,他可以晚上回家再複習一次。麻煩家長簽好請假單讓小明帶來,如果牙齒處理需要更多時間隨時跟我說!希望明天看牙齒順利!
```
**為什麼這樣寫**: 對應陳老師 style profile 主流風格 — 開頭「家長您好,沒問題的!」+ 引用具體事件(上午課程)+ 給 2 個 action items(午休補課 + 請假單簽收)+ 暖意收尾(希望順利)+ 帶 2-3 個驚嘆號。**predicted style match 89/100 是 3 個版本中最高**。

### Tone: **brief** (style match 72/100)
```
家長您好,沒問題!明天 10:30 進教室即可,我會請小明到座位上課,午休再幫他補上午重點。麻煩家長簽好請假單讓小明帶來,看牙齒順利!
```
**為什麼這樣寫**: 簡短版適合老師當天很忙時用。保留陳老師核心元素(暖意 + action items),但壓縮事件描述。風險:可能因太短失去陳老師「個人化」特色 — match 分較低(72)反映此點。

## 推薦版本: **warm**

老師,基於您過去 6 則回覆,**warm 版本最匹配您的個人風格**(match 89/100):您慣用「家長您好」「沒問題」開頭、會引用具體事件(上午第一節是國語、第二節是數學)、給 2-3 個 action items、結尾帶暖意。建議直接送出,或微調 1-2 字即可。

## Active Learning 結果

老師選擇 + 微調後送出版本:

> 家長您好,沒問題的!那小明明天 10:30 進教室,我會請他直接去座位坐好。上午第一節是國語、第二節是數學,我會在午休時找他 5 分鐘把上午重點補給他,他可以晚上再複習。麻煩家長簽好請假單讓小明帶來,如果牙齒處理需要更多時間隨時跟我說!

**Profile 更新**: n=6 → 7 樣本

**系統學到的變化(top 5)**:
- 具體行動建議增加 (1.33 → 1.57, Δ +0.24)
- 驚嘆號變少 (4.0 → 3.91, Δ -0.09)
- 回覆變更長 (103.33 → 105.43, Δ +2.10)
- 結尾更輕鬆 (1.25 → 1.07, Δ -0.18)
- 引用具體事件減少 (1.33 → 1.28, Δ -0.05)

### AI 解釋

陳老師,觀察到您這次又加了一個 action item「直接去座位坐好」,跟您一貫的「給家長具體可執行步驟」風格一致。下次系統會更傾向在草稿裡放 2-3 個 action items(您平均放 1.6 個),並維持「家長您好,沒問題的!」的開頭模式。整體風格穩定在 105 字左右、用 4 個左右驚嘆號、不用 emoji 但保持溫暖語氣。
