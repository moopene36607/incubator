# teachsay 老師家長 LINE 溝通助手報告

**模式**: 純函式 style + intent(免 API key)

## 老師基本資料

- **老師**: 陳老師
- **學校 / 班級**: 新北市某國小四年三班
- **歷史回覆 corpus**: 6 則

## 新進家長訊息

> 老師您好,小明明天因為要去看牙齒醫生(早上 9:30 預約),早上會晚一點到學校,大概 10:30 進教室。麻煩您讓小明先進來,我會跟他說要先看一下今天上課進度。謝謝!

**Intent 分類**: `leave_request` (請假)
**Urgency**: `high` / **Confidence**: 0.47

## Active Learning 模擬

假設老師選擇了某個草稿 + 微調,最終送出的版本是:

> 家長您好,沒問題的!那小明明天 10:30 進教室,我會請他直接去座位坐好。上午第一節是國語、第二節是數學,我會在午休時找他 5 分鐘把上午重點補給他,他可以晚上再複習。麻煩家長簽好請假單讓小明帶來,如果牙齒處理需要更多時間隨時跟我說!

*純函式模式無 AI 草稿生成。AI 模式會生成 3 個 tone(formal / warm / brief)+ 解釋。*
*teachsay 是輔助工具,**最終訊息送出前老師審閱**。重要案件(行為 / 投訴)建議親自完整撰寫。*
