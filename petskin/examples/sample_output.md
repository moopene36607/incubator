## ⚠️ LDA 模型假設與限制

- **常態分布假設**: 每類 features 假設 multivariate Normal 分布;真實有 skewed (搔癢分布偏右), Pro 版用 QDA / Mixture Discriminant
- **共同 covariance 假設**: LDA 假設所有類 covariance 相同;若不同則需 Quadratic Discriminant Analysis (QDA)
- **訓練樣本不大**: prototype 樣本小, real launch 需 ≥ 1,000 件多獸醫師標注 cases
- **特徵主觀**: 搔癢 / 紅腫 / 落毛 分數靠飼主肉眼判斷, 有 ±1-2 分主觀誤差, Pro 版加照片上傳 + vision 自動評估
- **不取代獸醫**: LDA 給「初步分流」是飼主決策輔助 (今天就醫 vs 觀察 24h vs 自行處理), **絕對不是診斷**;任何不確定都建議就醫
- **致命遺漏**: 不能分辨 **皮膚癌 / 自體免疫疾病 / 內分泌異常** (cushing / 甲狀腺低下) 引起的皮膚症狀, 這類嚴重病慢性病需獸醫鑑別
- **緊急紅旗永遠優先**: 發燒 / 食慾不振 / 嗜睡 / 嘔吐 / 全身性紅疹 → 不管 LDA 信心多高, 24h 內就醫

## 🤖 AI 獸醫師 triage 建議

陳小姐這個情況最大機率是跳蚤過敏 — 搬新家是常見觸發, 夏天活動的跳蚤可能藏在前住戶留下的地毯 / 沙發 / 木地板縫隙, 而屁股尾巴根部 (薦尾區) 是狗狗對跳蚤唾液過敏最典型發作位置, 強烈搔癢且伴隨少量皮屑符合這個 pattern。米克斯沒換食物排除食物過敏, 全身性而非僅耳朵也排除耳疥蟲。

今晚 / 明天可做的具體步驟: (1) **環境徹底處理** — 把所有寢具 / 沙發套 / 地毯用 60°C 以上熱水洗或丟掉, 噴環境用 IGR (Insect Growth Regulator) 殺幼蟲/卵, 不只殺成蟲。(2) **狗本身除蚤** — 去獸醫或寵物店買 Frontline Tritak / Bravecto 口服一錠 (大型犬 NT$800-1,200, 持效 3 個月), 不要用人類驅蟲產品。(3) **觀察 48-72 小時** — 搔癢應減半, 沒減就要 escalate。

就醫紅旗清單 (任一出現 24h 內衝急診): 搔抓到流血或自咬出大塊傷口、體溫超過 39.5°C、不吃飯 / 嗜睡 / 嘔吐、全身紅疹擴散到肚子內側、突然咳嗽呼吸急促。

風險提醒: LDA 是統計分流不是診斷, 三點重要 — (1) 如果除蚤 7 天沒改善, 一定要去獸醫排除「跳蚤過敏 + 異位性皮膚炎」共病 (60% 過敏狗有兩種以上致敏)。(2) 搬家壓力 + 環境改變也會誘發異位性皮膚炎發作, 兩者初期難分。(3) 黑狗皮膚變化不易觀察, 拍照記錄今天的狀態方便對比進度, 退色斑 / 結痂蔓延 / 皮膚變厚發黑都是惡化信號。
