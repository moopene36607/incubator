## ⚠️ SOM 模型假設與限制

- **2D 拓樸**: SOM 假設「相似 profile 距離近」是 2D 平面結構, 真實人格可能多維(SOM 強制壓平 → 邊界職涯可能不準)
- **訓練樣本量**: prototype 用 80 個 profile, real launch 需 ≥ 500 個各職涯實際工作 1-3 年滿意人士的 profile
- **Cold-start**: 新職涯類型 (e.g. AI 訓練師) 沒訓練樣本就找不到對應 cell, Pro 版需持續加新 profile
- **Feature 主觀**: 12 個自評 score 受主觀影響, Pro 版加客觀指標 (學測級分 / 程式作業 / 實習評價)
- **個性 ≠ 命運**: SOM 給「目前 profile 最像哪群人」, **不代表只能走那條路**;鄰近 cells 是合理的相近選項, 探索 mindset 比 dogmatic 更重要
- **不取代深度諮詢**: 大型決定 (轉行 / 換系 / 出國) 仍需學校輔導室 / 職涯顧問 / 業界 mentor 一對一輔導
- **隱私敏感**: 個人 profile 涉個資, 雲端版需匿名化 + 用戶同意 + 不販售給雇主

## 🤖 AI 職涯輔導 mentor 建議

陳同學, 你這個 profile 在 80 人 cohort 裡跟軟工那群最像 — 你的 analytical (8) + tech (7) + learning_speed (8) 是工程師核心配方, 而你比一般軟工 social (6 vs 平均 4) 和 communication (7 vs 平均 5) 都高, 這代表你「可以寫 code 也可以跟人講話」這個組合在台灣很值錢, 5-7 年後容易往 Tech Lead / 技術顧問 / Solution Architect 走, 比純軟工天花板高。SOM map 上你也離顧問 zone 不遠 (但距離 0.495+), 短期顧問業可能要硬補商業 case 訓練。

本月可做的探索動作: (1) **挑 1-2 個 side project** 用你資管背景做一個「給中小企業用的 SaaS 雛形」(POS 分析 / 報表自動化 / LINE Bot), 上傳 GitHub, 同時練軟工跟商業判斷;(2) **找 3 位 LinkedIn 上「資管系出身, 現在 5-10 年資歷的 軟工 / 解決方案架構師」做 30 分鐘訪談**, 用 EnglishGist / Calendly 排, 問「當初為什麼選軟工不選顧問」「現在後悔嗎」;(3) **試聽 1 場 BCG / McKinsey / 安永 在校園的徵才說明會** -- 不必投履歷, 看實際工作描述跟你想像的差距。

辨識訊號: (a) 持續 8 週每週投 5+ 小時在 side project 沒有 burnout = 真的喜歡寫 code; (b) 跟那 3 個 mentor 對話完比剛畢業還更興奮 = 真的好奇 / 不是想逃避找工作壓力。

風險提醒: 軟工起薪 (NT$45-75K) 比顧問 (NT$50-80K) 略低, 但 5 年後軟工易跳到 NT$120K+ 也容易遠距, 顧問則是 NT$100K+ 但平均 60-80 hr/週做 3-5 年才能升 Manager。SOM 是個性 mapping 輔助, 真實選擇仍需 2-3 次實習 + 多 mentor 訪談把訊號變強。
