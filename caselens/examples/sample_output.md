# caselens 法院判決書檢索報告

**模式**: 純函式 Stage 1 檢索 + Claude Stage 2 法律 re-rank

## 使用者情境

> 上週三晚上 10 點左右,我騎機車在台北市內湖區成功路三段正常綠燈直行,被一台白色 Toyota 汽車從後方追撞,我整個人飛出去摔在地上。當時我有戴安全帽。
>
> 事故當下我的機車後方車殼撞爛、後輪變形;我送急診照 X 光發現頸部揮鞭症候群、左手肘擦挫傷、左膝韌帶拉傷,休養 + 復健 2 個月期間沒辦法上班(我是工程師月薪 6 萬)。
>
> 對方說他當時在看 LINE 沒注意前方車流,但沒有酒駕也沒違規迴轉。對方有買強制險 + 商業險。
> 警方判定對方有未注意車前狀況之疏失。對方保險公司只願意理賠 NT$8 萬。
>
> 我想知道:1. 我這種情況法院通常會判多少賠償? 2. 對方保險公司給 NT$8 萬合理嗎? 3. 如果走訴訟我能拿到多少?需要多久?

**抽出的法律關鍵字**: 夜間, 復健, 戴安全帽, 揮鞭症候群, 擦挫傷, 機車, 汽車, 追撞, 酒駕

- **為什麼相似**: **幾乎是你的案件原型** — 同樣是機車正常行駛被汽車從後追撞、頸部揮鞭症候群、有戴安全帽、被告未保持安全距離(對應你案件中對方「看 LINE 沒注意前方」)、原告過失 0%。連賠償組成項目(醫療 + 工作損失 + 車輛維修 + 精神慰撫金)都跟你的案件吻合。**這個判例最值得用來談判和訴訟對標**。
- **法院**: 台北地方法院 / **判決日**: 2025-02-14
- **案件摘要**: 原告騎機車於外側車道正常行駛,被告駕汽車從後方追撞。原告頸部揮鞭症候群 + 機車後方損壞。原告有戴安全帽。
- **過失比例**: 原告 0% / 被告 100%
- **賠償金額**: NT$ 280,000
- **賠償組成**:
  - 醫療費: NT$ 65,000
  - 工作損失: NT$ 80,000
  - 車輛維修: NT$ 25,000
  - 精神慰撫金: NT$ 110,000
- **判決要旨**: 被告未保持安全距離為唯一過失。揮鞭症候群屬中度傷害,精神慰撫金 NT$11 萬。

### #2 `TCHV-114-A-015` — 汽車開門撞傷正常行駛機車
- **法律相關性**: 0.78
- **為什麼相似**: 雖然違規類型不同(開門 vs 追撞),但**法律核心一致**:「機車原告正常行駛 + 汽車被告違規 + 原告 0-5% 過失」。賠償金額 NT$51 萬可作為「**對方過失明確 + 中度傷害**」的上限參考(此案有鎖骨骨折比你嚴重,故偏高)。
- **法院**: 台中地方法院 / **判決日**: 2025-07-02
- **案件摘要**: 被告汽車路邊停車後突然開啟左側車門,撞擊正常行駛之原告機車。原告自摔受傷,鎖骨骨折 + 膝蓋擦傷 + 機車損壞。原告有戴安全帽。
- **過失比例**: 原告 5% / 被告 95%
- **賠償金額**: NT$ 510,000
- **賠償組成**:
  - 醫療費: NT$ 140,000
  - 工作損失: NT$ 200,000
  - 車輛維修: NT$ 30,000
  - 精神慰撫金: NT$ 140,000
- **判決要旨**: 被告開門未注意為主要過失(95%)。原告 5% 過失(未保持安全距離)。

### #3 `TCHV-113-A-010` — 汽車違規停車造成機車閃避不及撞傷
- **法律相關性**: 0.65
- **為什麼相似**: 同樣是「機車原告沒違規 + 汽車被告造成事故」,但傷勢較重(手腕骨折 vs 你的擦挫傷 + 揮鞭症候群)且本案原告有 20% 過失(閃避不及)。**精神慰撫金 NT$105K 跟 #1 接近**,可佐證揮鞭症候群類傷勢精神慰撫金實務區間。
- **法院**: 台中地方法院 / **判決日**: 2024-10-25
- **過失比例**: 原告 20% / 被告 80%
- **賠償金額**: NT$ 420,000

> **🚫 為什麼把 Stage 1 第二名 KSHV-114-A-008 排除**:該案是「**機車原告追撞前方計程車**」,角色方向跟你完全相反,法律意義不同(雖然 keyword 相似度高)。這是 LLM re-rank 的關鍵價值 — keyword 搜尋無法區分『誰撞誰』。

## 本案賠償區間預估

- **低估**: NT$ 230,000
- **中位**: NT$ 320,000
- **高估**: NT$ 450,000
- **依據**: 你的案件最接近 TPHV-114-A-011(NT$28 萬,揮鞭症候群 + 後追撞 + 原告 0% 過失)。但**你比該案多了「左手肘擦挫傷 + 左膝韌帶拉傷」**,屬於中度傷害,實務可能略高(精神慰撫金 NT$13-15 萬);加上工程師月薪 NT$6 萬 × 2 個月 = NT$12 萬工作損失也比該案 NT$8 萬高。整體預估區間 NT$23-45 萬,**中位 NT$32 萬**。

## 本案過失比例預估

- **預估原告過失**: 0%
- **預估被告過失**: 100%
- **依據**: 你綠燈直行 + 戴安全帽 + 警察筆錄判定對方未注意車前狀況有疏失 → 三個證據全部支持「對方 100% 過失」。最相似案件 TPHV-114-A-011 過失也是 0/100。**對方保險公司不能藉口減輕你的過失**。

## 🔼 加重 / 對你有利的因素

- 警方筆錄已判定對方有疏失(舉證責任轉到對方頭上)
- 你綠燈直行 + 戴安全帽(無法律可指責你的過失)
- 對方承認「看 LINE 沒注意前方車流」(自承過失證據)
- 揮鞭症候群屬「中度傷害」,精神慰撫金實務 NT$10-15 萬
- 工程師月薪 NT$6 萬有薪資證明 → 工作損失 NT$12 萬有憑可申報
- 對方有強制險 + 商業險 → 不會因對方賠不起而拿不到錢

## 🔽 減輕 / 對你不利的因素

- 揮鞭症候群難以「客觀」量化(法院通常依醫師證明天數判賠,不會完全照你主觀痛苦)
- 復健 2 個月實際是否每天就醫?復健記錄完整度影響精神慰撫金
- 韌帶拉傷的恢復天數需要醫師證明,否則對方會質疑 2 個月是否真的不能上班
- 機車是你自己的舊車,車輛維修費若車殼老舊,折舊後求償金額會被砍

## 蒐證清單

- ✅ 警察事故證明 / 道路交通事故初步分析研判表(必備)
- ✅ 對方筆錄影本(承認看 LINE 那段)
- ✅ 醫療診斷證明書(揮鞭症候群 + 擦挫傷 + 韌帶拉傷,寫明休養天數)
- ✅ 醫療收據(掛號 + 治療 + 復健全部)
- ✅ 復健治療記錄(每次就診 sign-off)
- ✅ 薪資證明 + 雇主開立的「請假無薪期間」證明(這項對工作損失最關鍵)
- ✅ 機車維修報價單 / 收據
- ✅ 事故現場照片(若有)+ 行車紀錄器(若有)
- ⚠️ 若可能,請對方寫一份「事實確認書」承認當時看手機(預防他後續翻供)

## 下一步實務建議

對方保險公司給 NT$8 萬**明顯偏低**(連工作損失 NT$12 萬都不夠)。建議分三步:

1. **先發存證信函**列明請求金額 NT$30-35 萬(中位數略高),附上述蒐證清單。給對方 14 天考慮期。
2. **若對方拒絕**,向你住所地的調解委員會(免費)申請調解。調解委員會通常會以類似 TPHV-114-A-011 案例為基準斡旋。1-2 個月內可結案。
3. **調解不成才走訴訟**。一審期程約 6-12 個月,律師費 NT$5-8 萬(可向對方求償)。**訴訟金額 NT$30 萬內,你的勝訴期望值仍高於 NT$25 萬**(扣除律師費後)。若金額 < NT$10 萬,可考慮小額訴訟程序自己跑(NT$1,000 裁判費 + 簡化程序)。

**避開的陷阱**:對方保險公司常用「先簽和解書才理賠 NT$8 萬」當話術,**簽下去你就失去後續追加請求權**。任何和解書簽之前一定要洽法扶基金會(412-8518 免費諮詢)或律師看過。
