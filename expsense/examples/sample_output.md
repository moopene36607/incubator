## ⚠️ Isolation Forest 模型假設與限制

- **無監督方法**: 沒有 labeled 訓練資料,只找「跟其他人不一樣」的點,**不等於詐騙**
- **小樣本 risk**: < 30 筆 IF 不穩定 (subsample 變小);Pro 版要求 ≥ 100 筆
- **特徵工程依賴**: 加更多 features (供應商 / 報銷時段 / 同筆多人會審) 可提高準確度
- **anomaly ≠ fraud**: 異常筆**只是值得問**,不是直接定罪;需要老闆 / HR 人工 follow-up
- **季節性 / 月底特例**: 業務員月底衝業績可能合理高消費,模型不知道 context
